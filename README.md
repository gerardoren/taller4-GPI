# Taller 4 – Gestión de Proyectos y Flujos de Trabajo Colaborativo

Este repositorio corresponde al Taller 4 del curso **Gestión de Proyectos de Investigación y Ciencia Abierta**.

El objetivo del taller es practicar:

- Gestión de proyectos con Quire
- Trabajo colaborativo con GitHub (ramas, conflictos y fusiones)
- Documentación de flujos de trabajo científicos
- Representación de procesos mediante Mermaid JS y Draw.io

## Flujo de trabajo

El flujo de trabajo del proyecto sigue tres etapas principales:

1. **Data Input**: generación o carga de datos.
2. **Data Processing**: limpieza y transformación.
3. **Data Analysis**: análisis estadístico y generación de resultados.

El diagrama formal del flujo se encuentra documentado en:
- `diagram.mermaid`
- Archivo `.drawio`
- Documento PDF de entrega
